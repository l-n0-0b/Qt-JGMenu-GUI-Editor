# Qt-JGMenu-GUI-Editor
Qt JGMenu GUI Editor is a small graphical configurator for [jgmenu] with menu collision checking written in Python 3 using PyQt6. Qt JGMenu GUI Editor — небольшой графический конфигуратор для [jgmenu] с проверкой конфликтов в меню, написанный на Python 3 с использованием PyQt6.
